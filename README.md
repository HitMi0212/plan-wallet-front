# Plan Wallet

`Plan Wallet`은 개인 수입/지출, 예적금/투자 흐름, 월별 통계를 관리하는 React Native(Expo) 가계부 앱입니다.

## 실행

```bash
npm install
npm run ios
```

또는 개발 서버 실행:

```bash
npm start
```

## 주요 기능

- 홈: 월 기준 요약, 금일 소비 확인, 거래 등록
- 거래: 달력 기반 조회, 거래 추가/수정/삭제, 목록 즉시 삭제
- 예적금/투자: 상품 및 입금 내역 관리
- 총재산/통계: 월별 집계 및 비교
- 카테고리: 수입/지출 카테고리 관리
- 설정: 로그인/로그아웃 및 데이터 관리

## 화면 동작 규칙

- 헤더 타이틀:
  - 홈: `YYYY년 M월`
  - 홈 외 메뉴: 각 메뉴명(거래/예적금·투자/총재산/통계/카테고리/설정)
- 빠른 현재 달 이동:
  - 거래/통계 화면에서 상단 월 텍스트를 누르면 현재 달로 즉시 이동

## 앱 이름

- 앱 표시 이름은 `Plan Wallet`로 설정되어 있습니다 (`app.json`).
