# Plan Wallet

`Plan Wallet`은 개인 수입/지출, 예적금/투자 흐름, 월별 통계를 관리하는 React Native(Expo) 가계부 앱입니다.

## 실행

```bash
npm install
npm run ios
```

또는 개발 서버 실행:

```bash
npm start
```

## 주요 기능

- 홈: 월 기준 요약, 금일 소비 확인, 거래 등록
- 거래: 달력 기반 조회, 거래 추가/수정/삭제
  - 예적금/투자 메뉴에서 생성된 연동 거래는 거래 메뉴에서 수정/삭제 불가
- 예적금/투자:
  - 상품 목록에서 상세 페이지 이동
  - 투자 상품은 증권사/화폐(KRW, USD) 선택 가능
  - 내역 추가 시 입금/수익/손실 선택 가능
  - USD 투자 내역은 거래/홈/총재산/통계에는 등록 시점 환율 기준 KRW로 반영
  - 메뉴 요약의 투자 총합은 KRW 기준으로 표시(USD 환산 포함)
- 총재산/통계: 월별 집계 및 비교
- 카테고리: 수입/지출 카테고리 관리
- 설정: 로그인/로그아웃 및 데이터 관리

## 환율 API

- 한국수출입은행 환율 API를 사용합니다.
  - URL: `https://oapi.koreaexim.go.kr/site/program/financial/exchangeJSON`
  - 기본 인증키는 코드에 설정되어 있으며, 필요 시 `EXPO_PUBLIC_KOREAEXIM_AUTH_KEY` 환경변수로 교체할 수 있습니다.

## 화면 동작 규칙

- 헤더 타이틀:
  - 홈: `YYYY년 M월`
  - 홈 외 메뉴: 각 메뉴명(거래/예적금·투자/총재산/통계/카테고리/설정)
- 빠른 현재 달 이동:
  - 거래/통계 화면에서 상단 월 텍스트를 누르면 현재 달로 즉시 이동
