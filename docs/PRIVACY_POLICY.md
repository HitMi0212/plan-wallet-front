# Plan Wallet 개인정보 처리방침

시행일: 2026-02-20

본 개인정보 처리방침은 Plan Wallet(이하 "앱")의 개인정보 처리 기준을 설명합니다.

## 1. 수집하는 정보

앱은 다음 정보를 사용자가 직접 입력한 경우에만 처리합니다.

- 회원 정보(선택): 이메일, 비밀번호, 닉네임
- 가계 데이터: 거래 내역(수입/지출, 금액, 카테고리, 발생일, 비고)
- 자산 데이터: 예적금/투자 상품 정보(은행명, 상품명, 입금 내역, 메모)
- 인증 정보: 로그인 상태 유지를 위한 토큰

## 2. 수집 및 이용 목적

- 사용자별 데이터 구분 및 앱 기능 제공
- 가계/자산 내역 저장, 조회, 통계 계산
- 백업 파일 내보내기 및 가져오기 기능 제공

## 3. 처리 방식 및 저장 위치

- 앱 데이터는 기본적으로 사용자의 기기 내부 저장소(로컬)에 저장됩니다.
- 개발/설정에 따라 외부 API 서버를 연동할 수 있으나, 기본 로컬 사용 시 서버 전송은 수행하지 않습니다.
- 백업 파일은 사용자가 직접 실행한 경우에만 생성/선택되며, 사용자가 선택한 외부 앱(파일 앱, 클라우드 등)으로 공유될 수 있습니다.

## 4. 제3자 제공 및 처리위탁

- 회사는 사용자의 개인정보를 제3자에게 판매하지 않습니다.
- 사용자가 백업 파일 공유 기능을 사용해 외부 서비스로 파일을 전송하는 경우, 해당 전송 이후의 데이터 처리는 해당 서비스의 정책이 적용됩니다.

## 5. 보관 및 파기

- 데이터는 사용자가 삭제하기 전까지 기기 내에 보관됩니다.
- 사용자는 앱 내 삭제 기능 또는 앱 삭제를 통해 데이터를 제거할 수 있습니다.
- 백업 파일은 사용자가 저장한 위치에서 직접 삭제할 수 있습니다.

## 6. 사용자 권리 및 행사 방법

사용자는 언제든지 아래 권리를 행사할 수 있습니다.

- 열람/정정: 앱 내에서 직접 조회 및 수정
- 삭제: 항목 삭제 또는 앱 삭제
- 동의 철회: 로그아웃, 데이터 삭제, 백업 파일 삭제

## 7. 아동의 개인정보

앱은 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 8. 보안

앱은 비인가 접근을 줄이기 위해 합리적인 보호 조치를 적용하려고 노력합니다.
다만, 기기 분실/탈옥/루팅 등 사용 환경에 따른 위험은 완전히 배제할 수 없습니다.

## 9. 처리방침 변경

본 처리방침은 법령, 서비스 또는 기능 변경에 따라 수정될 수 있으며, 변경 시 문서의 시행일을 업데이트합니다.

## 10. 문의처
